# Codo
Codo is a simple yet powerful terminal to-do list manager written in C.

## Installation

```bash
git clone https://github.com/Korbielowski/codo.git
cd codo
make
make install
```

### Dependencies
```bash
apt install sqlite3 libncursesw5-dev
```
